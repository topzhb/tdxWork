# 见龙在田自选股分析 — 评分算法说明

> 对应代码：`score.py`  
> 最后更新：2026-03-28

---

## 一、总体流程

```
候选池（板块筛选）
    ↓
技术面打分（满分100）
    ↓
基本面打分（满分100）
    ↓
综合得分 = 技术 × 60% + 基本面 × 40%
    ↓
最终得分 = 综合 × 70% + 板块热度加成（上限30）
    ↓
排序 → 输出 TOP N
```

## 二、技术面打分（满分 100 分）

### 2.1 趋势结构（30 分）

| 条件 | 得分 |
|------|------|
| MA5 > MA10 > MA20 > MA60（完整多头排列） | +15 |
| MA5 > MA20 > MA60（中期多头） | +8 |
| 收盘价站上 MA20 | +8 |
| MA60 斜率 > 1%（60线明显向上） | +7 |
| MA60 斜率 0~1%（60线微升） | +3 |

### 2.2 位置形态（25 分）

| 条件 | 得分 | 标签 |
|------|------|------|
| 52周位置 30%~70% | +15 | 位置适中 |
| 52周位置 70%~85% | +8 | 相对高位 |
| 52周位置 ≥ 85% | +2 | 高位注意 |
| 52周位置 < 30% | +5 | 低位 |
| 价格在布林中轨~上轨98%之间 | +10 | 布林中上轨 |

### 2.3 动量（25 分）

| 条件 | 得分 | 标签 |
|------|------|------|
| MACD 金叉（DIF 上穿 DEA） | +15 | MACD金叉 |
| MACD 多头（DIF > 0 且 DIF > DEA） | +12 | MACD多头 |
| RSI14 在 50~70（强势区） | +8 | RSI强势 |
| RSI14 ≥ 70（偏热） | +3 | RSI偏热 |
| RSI14 在 40~50 | +4 | — |
| KDJ J 在 20~80 且 K > D | +5 | KDJ多头 |

### 2.4 量能（20 分）

| 条件 | 得分 | 标签 |
|------|------|------|
| 近5日均量 > 近20日均量 × 1.5 | +12 | 近期放量 |
| 近5日均量 > 近20日均量 × 1.2 | +8 | 量能扩大 |
| 近5日均量 > 近20日均量 | +4 | — |

## 三、基本面打分（满分 100 分）

数据来源：**本地通达信 zip 优先 → 东方财富网络接口 fallback**

### 3.1 PE 估值（25 分）

| 条件 | 得分 | 标签 |
|------|------|------|
| 0 < PE < 20 | **+25** | PE低估 |
| 20 ≤ PE < 35 | **+18** | PE合理 |
| 35 ≤ PE < 60 | **+10** | PE偏高 |
| PE < 0（亏损） | **−5** | 市盈率为负 |
| 无数据 | 0 | 不计 |

### 3.2 净利润同比（35 分）⭐ 权重最高

> 口径：单季同比，来自通达信 `col[184]`，与东财数据完全一致

| 条件 | 得分 | 标签 |
|------|------|------|
| > +50% | **+35** | 净利爆发 |

| > +15% | **+20** | 净利增长 |
| > 0% | **+12** | 净利小增 |
| −10% ~ 0% | **−5** | 净利小降 |
| < −10% | **−15** | 净利大降 |

### 3.3 营收同比（15 分）

> 口径：单季同比，来自通达信 `col[183]`，与东财数据完全一致

| 市值范围 | 得分 | 标签 |

## 四、综合评分公式

```
综合得分  = 技术得分 × 0.60 + 基本面得分 × 0.40

最终得分  = 综合得分 × 0.70 + min(板块热度 × 2, 30)
```

**板块热度**来自热门板块算法（`t_daily_report` × `t_run_log`），反映该股所在板块在全市场中的活跃程度，最高贡献 30 分。

## 五、操作建议阈值

| 最终得分 | 建议 | 参考仓位 |
|------|------|------|
| ≥ 75 | 积极买入 | 15~20% |
| 65 ~ 74 | 关注买入 | 10~15% |
| 55 ~ 64 | 逢低关注 | 5~10% |
| < 55 | 观望 | 0% |

---

## 六、买入区间 / 止损 / 目标

```
买入区间 = 当日收盘价 × 0.97  ~  收盘价 × 1.02
止  损  = 收盘价 × 0.93  （−7%）
目  标  = 收盘价 × 1.15  （+15%）
```
