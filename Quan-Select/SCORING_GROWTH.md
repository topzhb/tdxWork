# 牛市成长股基本面分析策略

> 适用场景：牛市行情 + 国家政策扶持 + 行业景气上行 + 快速增长企业  
> 整理日期：2026-03-28

---

## 核心逻辑前提

牛市 + 政策驱动行情下，市场定价的是**预期溢价**，不是当期利润溢价。  
传统静态 PE、当期 ROE 等指标会**系统性失效**，需要换一套评价体系。

> 成长股的市场定价逻辑 = 未来两年的业绩，不是当期利润。

---

## 一、估值维度：抛弃静态 PE，用 PEG + PS

### 静态 PE 的局限

景气赛道成长股 PE 往往 40 ~ 100，静态打分体系会将其归入"高估"甚至扣分，  
但这恰恰是在定价未来高速增长，估值本身并不贵。

### ① PEG（PE ÷ 净利润增速）—— 成长股的"公允估值"标准

| PEG | 解读 | 建议评价 |
|-----|------|----------|
| < 0.5 | 严重低估，成长被大幅折价 | 满分 |
| 0.5 ~ 1.0 | 合理偏低，性价比高 | 高分 |
| 1.0 ~ 1.5 | 合理，市场充分定价 | 中分 |
| 1.5 ~ 2.0 | 偏贵，需景气持续验证 | 低分 |
| > 2.0 | 泡沫区，需谨慎 | 扣分 |

### ② PS（市销率）—— 适合利润尚未释放的早期成长股

> 适用场景：新能源、半导体、创新药早期——营收高增但净利润被研发/扩产压制

| PS | 行业参考 |
|----|----------|
| < 3 | 低估（偏制造业） |
| 3 ~ 8 | 合理（科技 / 消费） |
| 8 ~ 15 | 偏贵，需高增速支撑 |
| > 15 | 泡沫风险区 |

---

## 二、增速维度：增速加速度比绝对值更重要

### ① 增速加速度（二阶导）—— 最强买入信号

```
Q1 净利 +20%  →  Q2 净利 +35%  →  Q3 净利 +50%
                          ↑
              单季同比连续提速 = 景气拐点确认
              这个加速趋势的信号强度 > 静态的 +50%
```

**操作含义：** 当连续两期净利润增速提升时，是景气上行最确定的买入窗口。

### ② 收入 vs 利润增速的结构判断

| 组合 | 解读 | 评价 |
|------|------|------|
| 收入 +30% & 利润 +50% | 规模效应释放，利润弹性放大 | ✅ 最佳 |
| 收入 +30% & 利润 +10% | 扩张期，成本压制利润，需判断行业阶段 | ⚠️ 观察 |
| 收入 +5%  & 利润 +40% | 降本提效，非景气驱动，可持续性存疑 | ⚠️ 谨慎 |
| 收入 −5%  & 利润 +20% | 通常是卖资产或一次性收益，虚假繁荣 | ❌ 规避 |

---

## 三、ROE 维度：看趋势，不看绝对值

### 静态 ROE 的局限

快速扩张期的成长股大量融资、重资产投入，ROE 往往被稀释到 8 ~ 12%，  
但这恰恰是在"播种"，2 ~ 3 年后 ROE 会随产能利用率提升而爆发。  
用静态 ROE < 15% 扣分，会系统性低估优质成长股。

### ① ROE 趋势（比绝对值更重要）

- 连续 3 季度 ROE 持续提升 → 比当前绝对值高 20% 更有意义
- 融资摊薄期的 ROE 下降不应扣分，需结合净利润增速综合判断

### ② ROIC（投入资本回报率）—— 剔除杠杆的真实盈利能力

```
ROIC = EBIT × (1 - 税率) / (有息负债 + 股东权益)
```

> 对重资产成长企业（电力设备、半导体设备、新能源）尤为关键，  
> 能剔除财务杠杆干扰，反映资产本身的盈利效率。

| ROIC | 解读 |
|------|------|
| > 20% 且逐年提升 | 护城河加深，竞争优势稳固 |
| 15% ~ 20% | 优秀，值得重点关注 |
| 10% ~ 15% | 合格，需观察趋势 |
| < 10% | 资本回报偏低，慎重 |

## 四、景气度维度（核心补充指标）

**这是牛市政策驱动行情中最被低估的维度。**  
景气度决定"窗口期"，决定何时能享受戴维斯双击（业绩 + 估值同步提升）。

### 行业景气核心跟踪指标

| 指标 | 数据来源 | 景气信号 |
|------|----------|----------|
| 产能利用率 | 行业协会 / 季报 | > 85% = 供不应求，涨价可期 |
| 订单能见度 | 公司公告 / 调研 | 在手订单 / 年收入 > 1.5x = 高景气 |
| 产品价格走势 | 行情数据 | 量价齐升 = 景气顶部前最强信号 |
| 行业 PPI | 国家统计局 | 上游涨价向中游传导，毛利率改善 |
| 龙头公司毛利率 | 财报 | 毛利率持续提升 = 行业景气晴雨表 |

### 景气周期判断

```
景气底部（低产能利用率、低价格）→ 布局期
景气上行（产能利用率提升、价格上涨）→ 最佳持仓期  ← 核心窗口
景气顶部（产能过剩苗头、价格见顶）→ 减仓期
景气下行 → 清仓
```

## 五、政策驱动力度量化

### 三个层次，逐层递进验证

```
第一层：政策定调
    国家战略级（写入五年规划）> 部委专项规划 > 地方政策

第二层：资金落地
    专项债、产业基金、补贴、税收优惠是否明确落地

第三层：需求端验证（最关键）
    下游采购数据、装机量、招标量、出货量实际增长
```

### 最佳买入窗口

> **政策 → 资金 → 订单 → 业绩**，这条链条走到**"订单"**阶段时是最佳买入窗口。  
> 等到"业绩"完全兑现时，往往已在高位，性价比大幅下降。

| 阶段 | 市场状态 | 操作建议 |
|------|----------|----------|
| 政策定调 | 预期升温，板块启动 | 轻仓试探 |
| 资金落地 | 确定性提升，估值扩张 | 加仓 |
| 订单爆发 | 景气验证，戴维斯双击 | **重仓持有** |
| 业绩兑现 | 市场高度共识，拥挤交易 | 逐步减仓 |

## 六、综合打分体系改造建议

针对当前系统（`fund_score` 函数）的具体调整方向：

| 维度 | 现状 | 牛市成长股改进方向 |
|------|------|------------------|
| 估值 | 静态 PE 打分 | 改为 PEG；早期成长股用 PS 替代 |
| 净利增速 | 单期同比绝对值 | 增加连续提速判断（增速加速度） |
| 收入增速 | 单期同比绝对值 | 增加收入/利润结构比对分析 |
| ROE | 绝对值打分 | 改为 ROE 趋势 + ROIC |
| 市值 | 小盘加分 | 牛市初中期合理；尾期需降权 |
| **景气度** | **无** | **新增：订单/产能/毛利率趋势** |
| **政策强度** | **板块热度间接体现** | **可细化为政策层级权重** |

## 七、核心判断原则

**买入条件（同时满足）：**
1. 增速够快（净利润同比 > 30%，且在加速）
2. 景气未见顶（产能利用率上行，产品价格稳定或上涨）
3. 政策确定性强（资金已落地，订单已验证）
4. PEG < 1.5（成长性对估值有支撑）

**减仓 / 规避信号：**
1. 净利润增速开始放缓（即使绝对值仍高）
2. 行业产能利用率见顶，产品价格承压
3. 政策边际递减，资金关注度下降
4. PEG > 2，估值明显透支预期

---

> **一句话总结：**  
> 牛市成长股基本面核心只有三件事——**增速够快、增速在加速、景气还没见顶**。  
> 估值是次要的，因为市场在定价未来两年的业绩。  
> 当增速开始放缓、或行业产能利用率见顶，无论 PE 多低都要降低仓位。
